# Java Advanced Study（キム・ヨンハン／Inflearn ベース）

> Java の上級概念を整理し、実習コードを記録するリポジトリです。  
> 個人学習用であり、講座／プラットフォームとは直接の関係はありません。

## 目標
- 文字エンコーディング
- I/O の基礎と活用
- File, Files(パース表示、ファイル複製の最適化など)
- ネットワーク理論とプログラミング(IP、TCP、PORT、リソース整理 など)
- チャットプログラム(設計、クライアント、サーバー)
- HTTP とサーバ実装

## 環境
- JDK：**21**
- IDE：IntelliJ IDEA
